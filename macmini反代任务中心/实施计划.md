# Macmini反代任务中心 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 搭建一个可直接开始开发和本地验证的任务中心骨架，覆盖 API、4 槽位路由、Redis 队列、worker、Cloudflare Tunnel 与 Docker 隔离约束。

**Architecture:** 使用 `FastAPI + Redis + RQ` 实现控制层与执行层分离，4 个固定槽位独立排队。通过 `cloudflared` 将公网请求安全转发到本地 API，执行层按任务类型选择是否注入代理环境变量，同时避免影响现有 Docker 任务。

**Tech Stack:** Python 3.10, FastAPI, Pydantic, Redis, RQ, pytest, Docker Compose
